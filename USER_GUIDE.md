## 1. 🚀 เริ่มต้นใช้งาน

### 1.1 หน้า Welcome

เมื่อเปิดแอป SmartBrain Care ครั้งแรก จะพบหน้า Welcome ที่มีปุ่ม:
- **เข้าสู่ระบบ** — สำหรับผู้ที่มีบัญชีอยู่แล้ว
- **สมัครสมาชิก** — สำหรับผู้ใช้ใหม่

### 1.2 การสมัครสมาชิก (Register)

```
┌──────────────────────────┐
│     SmartBrain Care      │
│                          │
│  ┌────────────────────┐  │
│  │ Username           │  │
│  └────────────────────┘  │
│  ┌────────────────────┐  │
│  │ Password           │  │
│  └────────────────────┘  │
│  ┌────────────────────┐  │
│  │ Full Name          │  │
│  └────────────────────┘  │
│  ┌────────────────────┐  │
│  │ Phone              │  │
│  └────────────────────┘  │
│  ┌────────────────────┐  │
│  │ Birth Date         │  │
│  └────────────────────┘  │
│                          │
│  ┌────────────────────┐  │
│  │    สมัครสมาชิก      │  │
│  └────────────────────┘  │
└──────────────────────────┘
```

1. กดปุ่ม **"สมัครสมาชิก"** ที่หน้า Welcome
2. กรอกข้อมูล:
   - **Username** — ชื่อผู้ใช้ (ต้องไม่ซ้ำกับคนอื่น)
   - **Password** — รหัสผ่าน (ขั้นต่ำ 6 ตัวอักษร)
   - **ชื่อ-นามสกุล** — ชื่อจริงของคุณ
   - **เบอร์โทรศัพท์** — สำหรับติดต่อ
   - **วันเกิด** — วัน/เดือน/ปีเกิด
3. กดปุ่ม **"สมัครสมาชิก"**
4. ระบบจะพาไปหน้า Login อัตโนมัติ

### 1.3 การเข้าสู่ระบบ (Login)

1. กรอก **Username** และ **Password**
2. กดปุ่ม **"เข้าสู่ระบบ"**
3. ถ้าข้อมูลถูกต้อง จะเข้าสู่หน้าหลัก

> **💡 Tips**: ถ้าลืมรหัสผ่าน สามารถเปลี่ยนได้ที่หน้าตั้งค่า

## 🗂️ โครงสร้างแอป (Bottom Navigation)

แอป SmartBrain Care มี **4 แท็บหลัก** ที่แถบด้านล่าง:

```
┌──────────────────────────────────┐
│                                  │
│         (เนื้อหาหน้าจอ)          │
│                                  │
│  ════════════════════════════    │
│  🏠 หน้าแรก  💬 คำแนะนำ         │
│  👤 โปรไฟล์  📅 กิจกรรม         │
└──────────────────────────────────┘
```

| แท็บ | ไอคอน | หน้าที่ |
|------|-------|---------|
| **หน้าแรก** | 🏠 | แสดงสรุปข้อมูลคลื่นสมอง, สุขภาพ, Quick Actions |
| **คำแนะนำ** | 💬 | AI Chatbot ให้คำแนะนำด้านสุขภาพจิต |
| **โปรไฟล์** | 👤 | ข้อมูลส่วนตัว, แก้ไขโปรไฟล์, ตั้งค่า |
| **กิจกรรม** | 📅 | หมวดกิจกรรม, เกม, กำหนดการ |

## 2. 🧠 หน้าหลัก (หน้าแรก)

หน้าหลักแสดงข้อมูลสำคัญทั้งหมดในที่เดียว พร้อมการเชื่อมต่ออุปกรณ์ Muse S

```
┌──────────────────────────┐
│  📱 สวัสดี!               │
│  [ชื่อผู้ใช้]        ⚙️   │
│                          │
│  ┌────────────────────┐  │
│  │ 🔗 Muse S Headband │  │
│  │  [เชื่อมต่อ Muse S] │  │
│  └────────────────────┘  │
│                          │
│  ╔════════════════════╗  │
│  ║ คลื่นสมอง (EEG)    ║  │
│  ║                    ║  │
│  ║ Delta (0.5-4 Hz)   ║  │
│  ║ Theta (4-8 Hz)     ║  │
│  ║ Alpha (8-12 Hz)    ║  │
│  ║ Beta  (12-30 Hz)   ║  │
│  ║ Gamma (30+ Hz)     ║  │
│  ╚════════════════════╝  │
│                          │
│  ┌────────────────────┐  │
│  │ 📊 สรุปสุขภาพสมอง  │  │
│  │  PHQ-9: xx/27      │  │
│  └────────────────────┘  │
│                          │
│  [📋 ประวัติ] [📝 แบบทดสอบ]│
│                          │
│  ═══════════════════════ │
│  🏠  💬  👤  📅          │
└──────────────────────────┘
```

### 2.1 การเชื่อมต่ออุปกรณ์ Muse S

1. **เปิด Bluetooth** บนโทรศัพท์
2. **เปิดอุปกรณ์ Muse S** หรือ Muse 2
3. กดปุ่ม **"เชื่อมต่อ Muse S"**
4. ระบบจะค้นหาอุปกรณ์ → เลือกอุปกรณ์จากรายการ
5. รอจนแสดง **"เชื่อมต่อสำเร็จ"** (จุดเขียวส่องสว่าง)
6. หลังเชื่อมต่อสำเร็จ จะมีปุ่ม **"บันทึก"** เพื่อบันทึกข้อมูลคลื่นสมองลง Supabase

> **💡 ไม่มีอุปกรณ์ Muse?** กด **"ใช้โหมดจำลอง"** เพื่อทดลองใช้งานแอปได้เลย!

### 2.2 คลื่นสมองที่แสดง

| คลื่น | ช่วงความถี่ | ความหมาย |
|--------|-------------|----------|
| 🟣 **Delta** | 0.5-4 Hz | การนอนหลับลึก |
| 🟢 **Theta** | 4-8 Hz | ผ่อนคลาย / สมาธิ |
| 🔵 **Alpha** | 8-12 Hz | ตื่นตัว ผ่อนคลาย |
| 🟠 **Beta** | 12-30 Hz | คิดวิเคราะห์ |
| 🔴 **Gamma** | 30+ Hz | สมาธิสูง |

### 2.3 สรุปสุขภาพสมอง

การ์ดนี้แสดงผลจากแบบทดสอบ PHQ-9 ล่าสุดแบบ real-time (ดึงจาก Supabase) พร้อมแสดง:
- **ระดับสุขภาพจิต** — ปกติ / ซึมเศร้าเล็กน้อย / ปานกลาง / ค่อนข้างรุนแรง / รุนแรง
- **คะแนน PHQ-9** — xx/27 คะแนน
- **วันที่ทดสอบล่าสุด**

### 2.4 Quick Actions

ปุ่มลัดสำหรับเข้าถึงฟีเจอร์เร็วขึ้น:
- **📋 ประวัติ** — ดูประวัติผลทดสอบทั้งหมด
- **📝 แบบทดสอบ** — ทำแบบทดสอบ PHQ-9

## 3. 💬 AI คำแนะนำ (Chatbot)

แชทกับ **AI ผู้เชี่ยวชาญด้านสุขภาพจิต** พร้อมระบบ RAG ที่ให้คำตอบแม่นยำ

```
┌──────────────────────────┐
│  🤖 คำแนะนำ         ⚙️   │
│  ──────────────────────  │
│                          │
│  👤 คลื่น Alpha คืออะไร? │
│                          │
│  🤖 คลื่น Alpha (8-12Hz) │
│     เป็นคลื่นสมองที่เกิด │
│     ขึ้นเมื่อเราอยู่ใน    │
│     สภาวะผ่อนคลาย...     │
│     🔊 [ฟังเสียง]        │
│                          │
│  ──────────────────────  │
│  ┌────────────────┐ 🎤   │
│  │ พิมพ์ข้อความ... │ 📤   │
│  └────────────────┘      │
└──────────────────────────┘
```

### วิธีใช้ AI Chatbot

1. กดแท็บ **💬 คำแนะนำ** ที่ Bottom Navigation
2. **พิมพ์ข้อความ** ลงในกล่องข้อความด้านล่าง
3. หรือ **กดปุ่มไมค์ 🎤** เพื่อพูดสั่งด้วยเสียง (Speech-to-Text)
4. รอ AI ตอบกลับ (ใช้เวลาประมาณ 2-5 วินาที)
5. กดปุ่ม **🔊** ที่ข้อความ AI เพื่อฟังเป็นเสียง (Text-to-Speech)

### ตัวอย่างคำถามที่ถามได้

- "คลื่นสมอง Alpha คืออะไร?"
- "ฉันเครียดมาก ควรทำอย่างไร?"
- "แนะนำเทคนิคทำสมาธิหน่อย"
- "วิธีการนอนหลับได้ดีขึ้น"
- "สายด่วนสุขภาพจิตเบอร์อะไร?"

### ฟีเจอร์พิเศษ

| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| 🧠 **RAG System** | AI ค้นหาข้อมูลจากฐานความรู้ก่อนตอบ ทำให้คำตอบแม่นยำ |
| 📊 **Brainwave Context** | AI รับข้อมูลคลื่นสมองของคุณเพื่อให้คำแนะนำเฉพาะบุคคล |
| 🔊 **Text-to-Speech** | ฟัง AI ตอบเป็นเสียงภาษาไทย |
| 🎤 **Speech-to-Text** | พูดคำถามแทนพิมพ์ พร้อมแสดงสถานะ "กำลังฟัง..." |

> **💡 Tips**: ประวัติแชทถูกบันทึกใน Supabase — เปิดแอปครั้งต่อไปก็ดูประวัติการสนทนาเดิมได้

## 4. 👤 โปรไฟล์ (Profile)

หน้าโปรไฟล์แสดงข้อมูลส่วนตัวและเมนูต่างๆ

```
┌──────────────────────────┐
│      โปรไฟล์         ⚙️   │
│                          │
│       ┌────┐             │
│       │ 👤 │             │
│       └────┘             │
│     [ชื่อ-นามสกุล]       │
│     email@example.com    │
│                          │
│  ┌────────────────────┐  │
│  │ ข้อมูลส่วนตัว       │  │
│  │ ────────────────── │  │
│  │ 👤 ชื่อจริง-นามสกุล │  │
│  │ 📞 เบอร์โทรศัพท์    │  │
│  │ 📧 อีเมล           │  │
│  │ 🎂 วันเกิด          │  │
│  └────────────────────┘  │
│                          │
│  👤 แก้ไขข้อมูลส่วนตัว  ▸ │
│  🔖 บันทึกไว้           ▸ │
│  ⚙️ ตั้งค่า             ▸ │
│  ❓ ช่วยเหลือ           ▸ │
│  🚪 ออกจากระบบ          ▸ │
└──────────────────────────┘
```

### 4.1 ดูข้อมูลส่วนตัว

กดแท็บ **👤 โปรไฟล์** จะเห็นข้อมูล:
- **ชื่อจริง-นามสกุล**
- **เบอร์โทรศัพท์**
- **อีเมล**
- **วันเกิด**
- **รูปโปรไฟล์** (กดรูปเพื่อไปแก้ไข)

> **💡 Tips**: ดึงข้อมูลลงมา (Pull to Refresh) เพื่อโหลดข้อมูลล่าสุดจาก Supabase

### 4.2 แก้ไขข้อมูลส่วนตัว

1. กดเมนู **"แก้ไขข้อมูลส่วนตัว"** หรือกดที่รูปโปรไฟล์
2. ไปหน้า Edit Profile
3. แก้ไขข้อมูลที่ต้องการ (ชื่อ, เบอร์, อีเมล, วันเกิด, รูปโปรไฟล์)
4. กด **"บันทึก"**

### 4.3 เมนูในหน้าโปรไฟล์

| เมนู | คำอธิบาย |
|------|----------|
| 👤 **แก้ไขข้อมูลส่วนตัว** | ไปหน้าแก้ไขโปรไฟล์ |
| 🔖 **บันทึกไว้** | ดูรายการที่บันทึกไว้ |
| ⚙️ **ตั้งค่า** | ไปหน้าตั้งค่า (การแจ้งเตือน, รหัสผ่าน, ลบบัญชี) |
| ❓ **ช่วยเหลือ** | วิธีใช้งานแอป, ติดต่อเรา, สายด่วนสุขภาพจิต |
| 🚪 **ออกจากระบบ** | ออกจากระบบกลับไปหน้า Welcome |

## 5. 📅 กิจกรรม (Activities)

หน้ากิจกรรมรวบรวมฟีเจอร์ต่างๆ และระบบกำหนดการ

```
┌──────────────────────────┐
│       กิจกรรม        ⚙️   │
│                          │
│  ┌──────┐  ┌──────────┐  │
│  │👥     │  │🎮         │  │
│  │ผู้ดูแล │  │เกมคลาย   │  │
│  │      │  │เครียด    │  │
│  └──────┘  └──────────┘  │
│  ┌──────┐  ┌──────────┐  │
│  │🍽️     │  │⏰         │  │
│  │รู้     │  │จาก       │  │
│  │โภชนา │  │กิจวัตร   │  │
│  └──────┘  └──────────┘  │
│                          │
│  กำหนดการ        [+ เพิ่ม]│
│  ┌────────────────────┐  │
│  │ 09:00 ทดสอบความเครียด│  │
│  │ 12:00 พักผ่อนสายตา  │  │
│  │ 15:00 เกมบริหารสมอง │  │
│  └────────────────────┘  │
└──────────────────────────┘
```

### 5.1 หมวดกิจกรรม (4 หมวด)

| หมวด | ไอคอน | คำอธิบาย |
|------|-------|----------|
| **ผู้ดูแล** | 👥 | ข้อมูลผู้ดูแล, สายด่วนฉุกเฉิน, เบอร์ติดต่อ |
| **เกมคลายเครียด** | 🎮 | มินิเกมฝึกสมอง 4 เกม |
| **รู้โภชนา** | 🍽️ | อาหารบำรุงสมอง, เคล็ดลับโภชนาการ |
| **จากกิจวัตร** | ⏰ | กิจวัตรประจำวันแนะนำ แบบ timeline |

### 5.2 ผู้ดูแล (Caretaker)

แสดงข้อมูลผู้ดูแลและเบอร์ฉุกเฉิน:
- **รายชื่อผู้ดูแล** พร้อมเบอร์โทร (กดโทรได้เลย)
- **สายด่วนฉุกเฉิน**:

| หน่วยงาน | เบอร์โทร | เวลาทำการ |
|----------|----------|----------|
| 🆘 สายด่วนสุขภาพจิต | 1323 | 24 ชั่วโมง |
| 📞 สมาริตันส์ | 02-713-6793 | 24 ชั่วโมง |
| 🏥 ศูนย์ฉุกเฉิน | 1669 | 24 ชั่วโมง |

### 5.3 เกมคลายเครียด (Mini Games)

เกมฝึกสมอง **4 เกม**:

#### 🎯 เกมจับคู่ (Memory Game)
- **วิธีเล่น**: จำตำแหน่งการ์ดที่เปิด แล้วจับคู่ให้ถูกต้อง
- **ฝึก**: ความจำระยะสั้น, สมาธิ

#### 🌈 เกมจำสี (Color Sequence)
- **วิธีเล่น**: จำลำดับสีที่แสดง แล้วกดตามลำดับ
- **ฝึก**: ความจำลำดับ, สมาธิ

#### 🔢 ปริศนาตัวเลข (Number Puzzle)
- **วิธีเล่น**: เรียงตัวเลขให้ถูกลำดับ
- **ฝึก**: ตรรกะ, การวางแผน

#### ⚡ เกมปฏิกิริยา (Reaction Game)
- **วิธีเล่น**: กดปุ่มเร็วที่สุดเมื่อเปลี่ยนสี
- **ฝึก**: ความเร็วปฏิกิริยา

> **💡 เคล็ดลับ**: การเล่นเกมบริหารสมองวันละ 15-30 นาที ช่วยเสริมสร้างความจำและสมาธิได้ดี

### 5.4 รู้โภชนา (Nutrition)

แสดงข้อมูลอาหารที่ดีต่อสมอง:
- **รายการอาหารบำรุงสมอง** พร้อมประโยชน์และรายละเอียด
- **เคล็ดลับโภชนาการ** สำหรับสุขภาพจิตที่ดี

### 5.5 จากกิจวัตร (Daily Routine)

แสดงกิจวัตรประจำวันแนะนำแบบ Timeline:
- เรียงตามเวลาตั้งแต่เช้าจรดค่ำ
- แต่ละกิจวัตรมี emoji, ชื่อ, รายละเอียด และสีบอกหมวดหมู่

### 5.6 กำหนดการ (Schedule)

จัดการกำหนดการประจำวัน ข้อมูลบันทึกใน Supabase

#### วิธีเพิ่มกำหนดการ
1. ที่หน้ากิจกรรม เลื่อนลงมาส่วน **"กำหนดการ"**
2. กดปุ่ม **"+ เพิ่ม"**
3. กรอก:
   - **ชื่อกำหนดการ** — เช่น "ทำสมาธิ 10 นาที"
   - **เวลา** — เช่น "09:00"
   - **รายละเอียด** — คำอธิบายเพิ่มเติม
4. กด **"บันทึก"**

#### วิธีลบกำหนดการ
1. กดปุ่ม **🗑️** ที่ด้านขวาของรายการ
2. ยืนยัน **"ลบ"** ในหน้าต่างที่ปรากฏ

> **💡 Tips**: ดึงลงมา (Pull to Refresh) เพื่อโหลดกำหนดการล่าสุดจาก Supabase

## 6. 📝 แบบทดสอบ PHQ-9

แบบประเมินภาวะซึมเศร้ามาตรฐานสากล **PHQ-9** (Patient Health Questionnaire-9)

```
┌──────────────────────────┐
│  ← แบบประเมิน PHQ-9  ⚙️  │
│                          │
│  ข้อ 3 / 9         33%   │
│  ████████░░░░░░░░░░░░░░  │
│                          │
│  💡 ในช่วง 2 สัปดาห์     │
│     ที่ผ่านมา...          │
│                          │
│  ┌────────────────────┐  │
│  │ นอนไม่หลับ หรือ     │  │
│  │ หลับมากเกินไป       │  │
│  └────────────────────┘  │
│                          │
│  [0] ไม่เลย              │
│  [1] หลายวัน             │
│  [2] มากกว่าครึ่งหนึ่ง    │
│  [3] เกือบทุกวัน          │
└──────────────────────────┘
```

### วิธีเข้าทำแบบทดสอบ

1. จากหน้าหลัก กดปุ่ม **"📝 แบบทดสอบ"** ใน Quick Actions
2. อ่านข้อมูลเกี่ยวกับแบบทดสอบ PHQ-9
3. กดปุ่ม **"เริ่มทำแบบประเมิน"**
4. ตอบคำถาม **9 ข้อ** (ข้อละ 0-3 คะแนน)
5. เลือกคำตอบ → ระบบจะไปข้อถัดไปอัตโนมัติ
6. เมื่อตอบครบ ผลลัพธ์จะแสดงทันทีและบันทึกลง Supabase

### คำถาม PHQ-9 (9 ข้อ)

> **คำสั่ง**: _"ในช่วง 2 สัปดาห์ที่ผ่านมา คุณได้รับความเดือดร้อนจากปัญหาต่อไปนี้ มากน้อยเพียงใด"_

| ข้อ | คำถาม |
|-----|-------|
| 1 | รู้สึกเบื่อ ไม่สนใจอยากทำอะไร |
| 2 | รู้สึกหดหู่ ท้อแท้ หรือสิ้นหวัง |
| 3 | นอนไม่หลับ หรือหลับมากเกินไป |
| 4 | รู้สึกเหนื่อย ไม่มีแรง หรืออ่อนเพลีย |
| 5 | เบื่ออาหาร หรือกินมากเกินไป |
| 6 | รู้สึกไม่ดีกับตัวเอง คิดว่าตัวเองล้มเหลว |
| 7 | ไม่มีสมาธิในการทำสิ่งต่างๆ |
| 8 | พูดหรือเคลื่อนไหวช้าจนคนอื่นสังเกตเห็น หรือกระสับกระส่าย |
| 9 | คิดว่าตัวเองตายไปจะดีกว่า หรือคิดจะทำร้ายตัวเอง |

### ตัวเลือกคำตอบ (แต่ละข้อ)

| คะแนน | คำตอบ |
|-------|-------|
| 0 | ไม่เลย |
| 1 | หลายวัน |
| 2 | มากกว่าครึ่งหนึ่งของวัน |
| 3 | เกือบทุกวัน |

### เกณฑ์ประเมินผล (คะแนนเต็ม 27)

| ระดับ | คะแนน | คำอธิบาย | คำแนะนำ |
|-------|--------|----------|---------|
| 🟢 **ปกติ** | 0-4 | ไม่พบอาการซึมเศร้า | ดูแลสุขภาพกายใจเช่นเดิม |
| 🟡 **เล็กน้อย** | 5-9 | ซึมเศร้าระดับเล็กน้อย | ทำกิจกรรมที่ชอบ พูดคุยกับคนใกล้ชิด |
| 🟠 **ปานกลาง** | 10-14 | ซึมเศร้าระดับปานกลาง | ควรปรึกษาผู้เชี่ยวชาญ |
| 🔶 **ค่อนข้างรุนแรง** | 15-19 | ซึมเศร้าค่อนข้างรุนแรง | ควรพบจิตแพทย์โดยเร็ว |
| 🔴 **รุนแรง** | 20-27 | ซึมเศร้าระดับรุนแรง | กรุณาพบจิตแพทย์โดยด่วน สายด่วน 1323 |

> **⚠️ สำคัญ**: แบบทดสอบนี้เป็นเพียงเครื่องมือคัดกรองเบื้องต้น ไม่ใช่การวินิจฉัยทางการแพทย์  
> **📞 หากมีความคิดทำร้ายตัวเอง**: โทรสายด่วนสุขภาพจิต **1323** ตลอด 24 ชั่วโมง

```
┌──────────────────────────┐
│  ← ตั้งค่า                │
│                          │
│  🔔 การตั้งค่าการแจ้งเตือน ▸│
│  🔒 ตั้งค่ารหัสผ่าน       ▸│
│  🗑️  ลบบัญชี             ▸│
└──────────────────────────┘
```

### 8.1 การตั้งค่าการแจ้งเตือน

| การตั้งค่า | คำอธิบาย |
|-----------|----------|
| 🔔 **การแจ้งเตือนทั่วไป** | เปิด/ปิด Push Notifications |
| 🔊 **เสียง** | เปิด/ปิดเสียงแจ้งเตือน |
| 📳 **การสั่น** | เปิด/ปิดการสั่นเมื่อแจ้งเตือน |

### 8.2 ตั้งค่ารหัสผ่าน

1. กดเมนู **"ตั้งค่ารหัสผ่าน"**
2. กรอก:
   - รหัสผ่านเดิม
   - รหัสผ่านใหม่
   - ยืนยันรหัสผ่านใหม่
3. กด **"เปลี่ยนรหัสผ่าน"**

### 8.3 ลบบัญชี

1. กดเมนู **"ลบบัญชี"** (ตัวอักษรสีแดง)
2. อ่านคำเตือน: _"คุณแน่ใจหรือไม่? การลบบัญชีจะไม่สามารถย้อนกลับได้"_
3. กด **"ลบบัญชี"** เพื่อยืนยัน
4. ระบบจะลบข้อมูลทั้งหมดและกลับไปหน้า Welcome

> **⚠️ คำเตือน**: การลบบัญชีเป็นการลบถาวร ไม่สามารถกู้คืนได้

## ❓ คำถามที่พบบ่อย (FAQ)

### Q: ต้องมีอุปกรณ์ Muse หรือไม่?
**A:** ไม่จำเป็น! แอปมี **โหมดจำลอง (Simulation Mode)** ที่จำลองข้อมูลคลื่นสมองให้ทดลองใช้งานได้

### Q: AI Chatbot ฟรีหรือไม่?
**A:** การใช้ AI Chatbot ต้องใช้ OpenAI API Key ซึ่งมีค่าใช้จ่ายตาม usage

### Q: ข้อมูลปลอดภัยหรือไม่?
**A:** ข้อมูลถูกเก็บใน Supabase Cloud พร้อม Row Level Security ที่ป้องกันข้ามผู้ใช้

### Q: แบบทดสอบ PHQ-9 คืออะไร?
**A:** PHQ-9 (Patient Health Questionnaire-9) เป็นแบบประเมินภาวะซึมเศร้ามาตรฐานสากลที่พัฒนาโดย Drs. Robert L. Spitzer, Janet B.W. Williams และ Kurt Kroenke ใช้คัดกรองเบื้องต้น 9 คำถาม คะแนนเต็ม 27

### Q: แอปรองรับภาษาอะไรบ้าง?
**A:** ปัจจุบันรองรับ **ภาษาไทย** เป็นหลัก รวมถึง AI ตอบเป็นภาษาไทย

### Q: สามารถใช้บนอุปกรณ์อะไรได้บ้าง?
**A:** รองรับ Android, iOS, macOS, Windows, Linux, และ Web
